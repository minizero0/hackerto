## Development Step

  **1. source shev/Scripts/activate** 가상환경 작동

  **2. cd easy_volunteer** 프로젝트 폴더로 이동

  **3. pip install django** 장고 설치

  **4. pip install pillow** 동적 이미지를 위한 pillow  설치

  **5. python** **manage.py runserver** 서버 작동
